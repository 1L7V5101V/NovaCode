## 1. 什么是 `json.loads()`？

JSON 是一种我们在编程中常用的、有严格格式要求的**数据传输格式**（比如必须有花括号 `{}`，键值对必须用双引号 `""` 包裹）。

而 `json.loads()` 是 Python 语言自带的一个**标准工具函数**。它的唯一功能，就是去检查一段文本到底是不是符合规范的 JSON。

它的工作特点可以用三个词形容：**速度极快、不讲人情、零成本**。

- **如果格式完美**：它在 0.00001 秒内就能通过。
    
- **如果格式有错**：它会立刻**报错（抛出异常）**，并且明确告诉你错在哪里（比如：第 3 行少了一个逗号）。
