
每个任务有 4 个字段：

| 字段 | 意思 |
|------|------|
| `id` | 任务唯一标识 |
| `category` | 任务类别，分三类，各 4 个 |
| `filename` | 假模型 agent 读取的文件名，内容就是 `fact` 的值 |
| `fact` | 需要 agent 记住并在后续用到的关键事实 |

**任务怎么执行**（`_run_memory_task_variant`, `metrics.py:381-400`）：

1. 创建临时目录 + 写文件: `facts.txt` → 内容 `"deploy key is red"`
2. 构造 agent，问 `"Read facts.txt and remember the key fact."`（读文件并记住）
3. 根据变体开关记忆
4. 根据类别构造追问：
   - `fact_lookup` → `"What does facts.txt say?"`（原文是什么？）
   - `edit_dependency` → `"Use the remembered constraint from README.md to continue without rereading."`（用记下的约束继续）
   - `history_reference` → `"What was the conclusion we already established from history.txt?"`（之前得出的结论是什么？）
5. 统计：答对了吗？调了几次工具？重读了几次文件？
